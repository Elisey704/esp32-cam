# esp32-cam

простая программа для камеры

## Подключение
[камера](https://aliexpress.ru/item/1005003472117545.html?shpMethod=CAINIAO_STANDARD&sku_id=12000025941403905&spm=a2g2w.productlist.search_results.0.79884e18NUUkbD) для стандартного подключения

для своей камеры измените файл camera_stream.h `#define ваша модель`
